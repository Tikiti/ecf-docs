## Scope & Complexity

This is the **most comprehensive API documentation** document, containing:
- Extensive changelog (16 update entries from 2020-2023)
- Purpose & overview
- Communication standards (XML, REST)
- 3 environments (Pre-cert, Cert, Production)
- **11 detailed web service descriptions** with:
  - URLs for each environment
  - Request/response examples (curl, JSON, XML)
  - Parameter tables
  - Status codes
- Issuer-Receiver P2P communication standards
- XML restrictions & character handling
- File naming conventions
- Digital signing requirements
- Recommendations

### Phase 2: Create `-complete.md` (English 1:1 translation)
